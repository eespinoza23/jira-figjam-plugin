🔌 Jira Multi-Import for FigJam

This is a FigJam plugin backend serving:

  • manifest.json — FigJam plugin manifest
  • /api/jira-* — Jira OAuth & search endpoints
  • /public/* — Plugin frontend (code.js, ui.html, icons)

Usage: Load this plugin in FigJam using the manifest URL:

https://jira-figjam-plugin.vercel.app/manifest.json
